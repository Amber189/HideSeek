HideSeek

A simple, yet customizable live search jQuery plugin

download or view on GitHub

HideSeek is yet another jQuery plugin for tabs, is compatible with IE7+, Chrome, Firefox, Safari and jQuery 1.8.1, 1.9.1, 1.10.1, 2.0.2. and supports custom events, rotation and animation
